# crate2nix

crate2nix generates [nix](https://nixos.org/nix/) build files for [rust](https://www.rust-lang.org/) crates
using [cargo](https://crates.io/).

➡️ [documentation](https://nix-community.github.io/crate2nix/) ⬅️
